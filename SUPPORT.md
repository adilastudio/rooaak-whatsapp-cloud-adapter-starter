# Support Policy

## Scope

This repository is maintained by Rooaak as an official WhatsApp Cloud adapter starter.

## What we support

1. Bug fixes for documented Meta webhook and Rooaak webhook flows.
2. Security fixes for critical issues.
3. Compatibility updates for Rooaak API/SDK contract changes.

## Response expectations

- Best-effort support via GitHub Issues.
- No SLA is provided for community support requests.

## Out of scope

1. Custom Meta Business verification or policy-review workflows.
2. Production incident response for third-party infrastructure.
3. Long-term support branches.
